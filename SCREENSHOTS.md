# Screenshots Guide

Since we can't upload actual screenshots to GitHub without images, here are descriptions of each screen:

## Dashboard
- Center: Three stat cards showing Total Students, Unpaid Bills, Total Outstanding
- Top: Charts showing bill status distribution and monthly collections
- Bottom: Quick actions and recent unpaid bills table

## Student Management
- Table view of all students with Name, Grade, Guardian, Phone
- "Add New Student" button at top
- Each row has View and Delete buttons

## Bill Creation
- Form with fields: Student (dropdown), Description, Amount, Due Date
- "Create Bill" and "Cancel" buttons
- Shows list of all students for selection

## Payment Processing
- Form to select bill, enter amount, choose payment method
- Shows list of unpaid bills with balances
- "Record Payment" button

## Reports
### Outstanding Report
- List of students with outstanding payments
- Shows total amount due per student
- Details of each unpaid bill
- "Export as PDF" button

### Collections Report
- Today's collections total
- This month's collections total
- Table of recent payments
- "Export as PDF" button

## Authentication
### Login Page
- Clean form with username and password fields
- School Billing System header
- Default credentials note

### User Menu (Logged in)
- Shows username and role in top right
- Logout button
- Navigation menu with all features

## Features in Action
1. **Bulk Bill Creation**: Checkbox interface to select multiple students
2. **Search Functionality**: Search bars for bills and students
3. **PDF Export**: Professional formatted PDF downloads
4. **Charts**: Interactive doughnut and bar charts
5. **Mobile Responsive**: Adapts to phone and tablet screens

## Color Scheme
- Primary: #2c3e50 (dark blue)
- Success: #27ae60 (green)
- Warning: #f39c12 (orange)
- Danger: #e74c3c (red)
- Info: #3498db (blue)
